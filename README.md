# exercicio_sql_banco_de_dadps
SQL

Eu fiz um exercício criando uma tabela chamada LIVRARIA, onde cadastrei informações de livros, autores, editoras e ano de publicação. Depois inseri dez livros diferentes para montar o banco de dados. Em seguida, rodei algumas consultas para treinar: busquei todos os dados, selecionei autores com suas editoras, filtrei livros por sexo do autor, peguei valores de livros publicados em São Paulo e também ajustei a consulta que buscava autores masculinos com livros publicados em SP ou RJ, usando o operador OR.

No final, ainda criei uma tabela simples chamada CLIENTE e cadastrei meu próprio nome, endereço e telefone.
